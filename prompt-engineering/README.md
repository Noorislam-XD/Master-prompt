# ⚙️ Advanced Prompt Engineering Techniques

Universal techniques that work across all LLMs — based on academic research and the [DAIR.AI Prompt Engineering Guide](https://www.promptingguide.ai/). These techniques are model-agnostic and represent the state of the art in prompt design.
