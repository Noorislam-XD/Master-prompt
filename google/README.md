# 🔵 Google / Gemini Prompt Packs

Prompts based on Google's official [Gemini Prompting Guide](https://ai.google.dev/gemini-api/docs/prompting-strategies), the [Google Prompt Engineering Whitepaper](https://www.kaggle.com/whitepaper-prompt-engineering), and the [Google Cloud Vertex AI prompt documentation](https://cloud.google.com/vertex-ai/generative-ai/docs/learn/prompts/introduction-prompt-design).

## Prompts in this folder

| File | Technique | Use Case |
|------|-----------|----------|
| [`step-back.md`](./step-back.md) | Step-Back Prompting | Abstract to principles before solving specifics |
| [`self-consistency.md`](./self-consistency.md) | Self-Consistency | Generate multiple answers and take the majority |
| [`least-to-most.md`](./least-to-most.md) | Least-to-Most | Decompose complex problems into simpler sub-problems |
| [`multimodal.md`](./multimodal.md) | Multimodal Prompting | Combine text and image inputs effectively |
| [`system-instruction.md`](./system-instruction.md) | System Instructions | Structure Gemini system prompts for consistent behavior |
| [`zero-shot-cot.md`](./zero-shot-cot.md) | Zero-Shot CoT | Trigger reasoning with "Think step by step" |
